Open HTTP.properties.php
change $APJP_KEY to key of your choice
Upload to server

On client side, set server address to http://your_server.com/your_path/HTTP.php
and set the key to be the same as your server's key
choose APJP compat mode for server encryption!

This pack is a redistribution of the APJP server side (the 'p' side = =||)

====
Chinese (Simpl.)
====
在记事本打开 HTTP.properties.php
把 $APJP_KEY 变成你希望的钥匙
把所有文件上传到服务器

在客户端，添加服务器
设置服务器地址为 http://你的服务器/KnProxy目录/HTTP.php
设置服务器钥匙与服务器相同
在加密模式中选择 [x] 兼容模式（兼容APJP）

注意，这个只是 APJP 的服务器端的再发行！如果你有架设好的 APJP 服务器，可以直接用！
